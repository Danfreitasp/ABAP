Programas da academia ABAP.

Exercício 1: Importação de Dados Contábeis para Legado - Especificação Funcional
Descrição: Este projeto tem como objetivo criar uma solução de importação de dados contábeis para um sistema legado usando o SAP S/4HANA. 
A solução permitirá a importação de lançamentos contábeis a partir de um arquivo externo no formato CSV,
validará os dados e registrará os lançamentos contábeis correspondentes no sistema.

Exercício 3: Criação de Aplicação SAP Fiori para Consulta de Pedidos de Compras
Enunciado: Você foi designado para criar uma aplicação SAP Fiori que permita aos usuários consultar pedidos de compras, 
realizar somatórias dos itens desses pedidos e aplicar filtros para facilitar a busca. 
A aplicação deve oferecer uma experiência de usuário intuitiva e eficiente. 

Exercício 4: Interface de Criação de Bancos - Especificação Funcional
Descrição: Este projeto visa criar uma interface que permitirá a criação de novos bancos no sistema SAP S/4HANA a partir de um arquivo texto. 
A interface utilizará a transação FI01 para criar os bancos, empregando técnicas avançadas de programação ABAP, como o uso do batch input. 
Além disso, será gerado um relatório de processamento, e os bancos criados com sucesso poderão ser acessados pela transação FB03.
